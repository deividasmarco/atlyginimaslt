# Logo Setup Instructions

Save the attached logo image to these paths (replace the placeholder files):

1. `assets/icon.png` — 1024×1024 px (the attached image is already the right size)
2. `assets/adaptive-icon.png` — same image, 1024×1024 px  
3. `assets/splash.png` — same image works, or a version without the "LT" text at 1284×2778 px

The app.json already references these paths, no changes needed there.

After saving, run: `npx expo start --clear`
